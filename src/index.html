Implementation Notes

WARNING!
This endpoint permanently deletes the budget from the storage facility. The deletion is permanent, and unrecoverable. There is no undo. Generally, your implementation should close a budget, rather than deleting it.

Responses

STATUS 200
The budget was deleted.
STATUS 401
Returned if the user is authenticated, but lacks the permission to view the requested fields.
STATUS 404
Returned if the user is not authenticated, or the supplied {id} does not exist.
